Like-predicate

We used the same pattern of like in searchResults.java for title, year, director, firstname, and lastname. Search Pattern: '%AN%': All strings that contain the pattern 'AN' anywhere. For example, 'LOS ANGELES' and 'SAN FRANCISCO' would both satisfy the condition. For Browse.java, we used a like operator on the title name starting with an alphanumeric character. Browse Pattern: 'ABC%': All strings that start with 'ABC'. For example, 'ABCD' and 'ABCABC' would both satisfy the condition.
